# 🧠 Memory Kit

**Your Heyron agent forgets everything between conversations. This fixes that.**

---

## What This Does

Your agent remembers:
- Who you are
- What you've worked on
- Your preferences
- What's important to you

No scripts. No Python. No complexity.

---

## How To Install

1. **Copy these files to your agent's workspace:**
   - `USER.md` — details about you
   - `SOUL.md` — how your agent behaves
   - `HEARTBEAT.md` — scheduled tasks

2. **Edit USER.md** — fill in your name, preferences, what you care about

3. **Edit SOUL.md** (optional) — customize how your agent acts

That's it. Done.

---

## What Each File Does

| File | What It Is |
|------|------------|
| `USER.md` | Everything about YOU — name, email, preferences, projects |
| `SOUL.md` | Your agent's personality and rules |
| `HEARTBEAT.md` | Things your agent checks regularly |

---

## Why This Works

When your agent starts a conversation, it reads these files first. Now it knows:
- Who you are
- What you care about
- What you've been working on

That's memory. Simple files, big difference.
