# 🧠 Memory Kit

**Your Heyron agent forgets everything between conversations. This makes it remember.**

---

## What This Does

- Your agent can now answer: "What did we work on last week?"
- It searches through all your files instantly
- Important stuff shows up first
- Costs $0 — no extra services needed

---

## Install (3 steps)

1. **Copy the files** — Download this repo or clone it
   
2. **Run setup** — In your chat, tell your agent:
   ```
   chmod +x setup.sh && ./setup.sh
   ```

3. **Build the memory** — Tell your agent:
   ```
   python3 scripts/SYSTEM/memory_index.py --build
   ```

That's it. Done.

---

## How To Use It

Just ask your agent questions like:

- "What did we do yesterday?"
- "Show me our projects"
- "What have we been working on?"

Your agent will search through all your files and FIND the answer.

---

## What You Need

- A Heyron container with Python 3
- That's it. No API keys. No accounts.
