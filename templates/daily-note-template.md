# Template: Daily Session Notes

Copy this to `memory/daily/YYYY-MM-DD.md`:

```markdown
# YYYY-MM-DD - Daily Notes

## What We Did
- [What you worked on]

## Decisions Made
- [Key decisions]

## Next Steps
- [What to do next]

## 1% Growth
- [One improvement from today]

## Other Notes
- [Anything else worth remembering]
```

---

## Tips

- One file per day
- Use the date format so it's easy to search
- Include decisions and next steps
- The "1% Growth" is what made you 1% better today
