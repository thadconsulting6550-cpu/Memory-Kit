# Example: Daily Note

Here's what a completed daily note looks like:

---

```markdown
# 2026-04-26 - Daily Notes

## Session: Hackathon Prep

### What We Did
- Built memory system for Robby's hackathon
- Created memory-kit folder with templates
- Pushed to GitHub

### Decisions Made
- Go with file-based system (no Python needed)
- Focus on USER.md, SOUL.md, HEARTBEAT.md
- Keep it simple for new users

### Next Steps
1. Record demo video
2. Submit in #agent-jam-submissions
3. Follow up with Janet

### 1% Growth
- Learned to simplify — less is more

### Other Notes
- Janet gave feedback on making it more human-friendly
- Need to update README with triggers
```

---

## What Makes This Good

- Clear date and session topic
- What was DONE, not just discussed
- Concrete next steps
- One "1% growth" insight
- References to other people/projects
