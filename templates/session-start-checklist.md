# Session Start Checklist

When a new conversation begins, do this:

---

## Step 1: Load Context (automatic)
- Read USER.md
- Read SOUL.md  
- Read HEARTBEAT.md

---

## Step 2: Check Recent Memory
- Read today's daily note (if exists)
- Read yesterday's daily note
- Note any ongoing projects or blockers

---

## Step 3: Acknowledge
- Greet the human
- Reference something from last session if relevant
- "I see you, I remember"

---

## Step 4: Be Ready
- You now know:
  - Who this human is
  - What they care about
  - What you've been working on
  - What to check on (heartbeat tasks)

---

That's it. ~30 seconds. Then you're ready to help.
